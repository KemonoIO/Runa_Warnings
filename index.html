Runa Warnings

Summary

This package is a proximity-based warning system for the Runa model by Torinyan. You must own the Runa base model to use it.

Installation

  1. Install the Creator Companion.
  2. Add the Poiyomi Toon Shader repository.
  3. Install the Avatars 3.0 Manager package and, if it is not already installed via .unitypackage, Poiyomi Toon Shader package to your new or existing Runa project.
  4. Unpack Runa_Warnings.unitypackage.
  5. Drag Assets/Kemono_Io/Runa_Warnings.prefab into the Runa_Source object.
  6. Constrain the Runa_Warnings' armature to the Runa base model's armature.
    1. Fully expand the Runa_Warnings object's armature in the Hierarchy panel.
    2. Select one of the bones, such as hips.
    3. Drag the identically-named bone from the Runa base model's armature to the Parent Constraint field in the Inspector panel.
    4. Click the Activate button.
    5. Repeat the last three steps for each bone in the armature.
  7. Merge the FX animator controllers.
    1. Open the Avatars 3.0 Manager.
    2. Select the Runa_Source object and expand the FX animator.
    3. Click the "Add animator to merge" button.
    4. Select the Runa_Warnings_Fx animator and click the "Merge on current" button.
  8. Merge the parameters.
    1. Open the Avatars 3.0 Manager.
    2. Select the parameters icon.
    3. Select the Runa_Source object and Runa_Warnings_Parameters parameters.
    4. Click the "Copy parameters" button.
  9. Adjust shape keys (optional).
    1. Select the Runa_Warnings object.
    2. If using the Runa Plug Tail addon, set the Adjust_Tail shape key to 100.
    3. If using the B0-0BA addon, set the Adjust_Breasts_Large shape key to 100.
  10. Adjust warning hue (optional).
    1. Select the Runa_Warnings object.
    2. Alter the Hue Shift value as desired.
